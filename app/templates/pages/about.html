About

Hey Gurgle is a search engine for the Marsupial Gurgle audio clip archive for the podcast, TBTL: Too Beautiful to Live and born out of the need for a better searching mechanism for specific audio clips.

Marsupial Gurgle is uses WordPress for its content management system and every clip is catalogued blog posts or standalone pages, or both. Searching for a clip means that the search results would be for blog posts or pages, and not the clips individually.

As part of a larger project, each of the audio clips uploaded to the backing storage for Marsupial Gurgle has been re-indexed and the metadata stored in a database where full-text searching can be done for individual clips.

The Hey Gurgle Name

The Hey Gurgle name was chosen to represent the Marsupial Gurgle website and audio clip archive, along with the "Gorgle" spoof on Google's voice assistant wake up phrase and its proximity to the word "gurgle".

At various points in the podcast, Luke Burbank or Andrew Walsh have said Hey, Gorgle! dozens of times. It made sense to name the Marsupial Gurgle audio clip search engine, Hey Gurgle.

Technical Details

The search engine is written in Python, built using the Flask web application framework for the backend and Bootstrap toolkit for the frontend. The database with the audio clip metadata runs on MySQL Server and uses the included full-text searching feature and the natural language and boolean search modes.

To parse through and re-index all of the uploaded audio clips, the Marsupial Gurgle Audio Clip Parsing Tool was creating in Python, pull audio tag metadata and parse the file paths for all clips, then store the data into the aforementioned MySQL database.
